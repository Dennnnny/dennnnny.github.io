今天來建立Gatsby部落格
2022-10-11
圖片來源 unsplash

架起你的部落格

我是採用 Gatsby 的 Quick start
首先要確定自己的 node 版本有在 14.15 以上
不然啟動伺服器的時候會遇到打不開的問題

使用命令提示字元(cmd) 來建立這次的專案 用指令快速建起一個 Gatsby 專案
npm init gatsby
輸入完後 照著題目選擇你想要的語言/插件/額外功能：

我的設定檔如下：

What would you like to call your site? // 取名字
✔ · My Gatsby Site
What would you like to name the folder where your site will be created? // 取資料夾名稱
✔ /my-gatsby-site
✔ Will you be using JavaScript or TypeScript? // 選擇 js || ts
· JavaScript
✔ Will you be using a CMS? // 需不需要額外的內容管理系統
· No (or I'll add it later)
✔ Would you like to install a styling system? // 需要用什麼樣式套件
· styled-domponents
? Would you like to install additional features with other plugins? // 多選 可以選GA, sitemap, md/mdx
(Multiple choice) Use arrow keys to move, spacebar to select, and confirm with an enter on "Done"
❯◯ Add the Google Analytics tracking script
 ◯ Add responsive images
 ◉ Add an automatic sitemap
 ◯ Generate a manifest file
 ◉ Add Markdown and MDX support
 ◯ Add Markdown support (without MDX)

我選了 js (後面會提我把 js 轉成 ts 的過程)
樣式選擇 styled-components
沒有使用 CMS 而是直接用 md 檔， 所以加裝了 md/mdx support

完成後 你的網站就可以跑起來了 ～
在 cmd 中 把本地端伺服器架起來
npm run develop
然後開啟瀏覽器 輸入網址： http://localhost:8000
就可以看到你的網站了 🎉
之後就可以藉由編輯 src/pages/index.js 這個檔案
然後就可以馬上在畫面上看到更新
