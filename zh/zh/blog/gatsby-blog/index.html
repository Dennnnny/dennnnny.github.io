2022-09-13
圖片來源 unsplash

開始用 Gatsby 來架我的部落格

之前試著用 hexo 來架部落格
可以在短時間裡就把一個網站架好
真的是很快

主題也有滿多可以在官網裡找
或者別人寫好的主題可以直接套來用
不過 有時候就是
覺得有些主題裡總是差了一點功能 但自己要調整的話 也是滿費勁的

後來發現 Gatsby
覺得這套更有彈性
對我來說比較容易新增自己想要的功能
雖然彈性就代表著自己會花更多心力
不過我很享受 創造的過程

雖然也有其他別的 React 相關框架可以當成部落格的
我想 next.js 或 remix.js
都可以很快地架出部落格 也有很多強大的功能
不過 覺得自己的部落格其實只需要靜態的呈現就足矣
所以就決定要使用 Gatsby 了！！

之後會慢慢補一些在建立專案的時候 遇到的一些困難 😂
以上是今天的分享 👋
