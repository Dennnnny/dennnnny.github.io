我把Google Sheet當成Cloud Function用
2022-12-12
圖片來源 unsplash

來使用 google 試算表執行程式

今天來分享一下 最近做了一個提醒的通知機器人 機器人的部分其實只是串接了 slack 的 webhooks 在以前我通常做這種排程通知 大多都選擇使用 google 的 cloud function 不過因為免費試用有期限 後續繼續使用的話需要收費 所以我這邊改用 google 的 spread sheet 來做這件事情

使用者故事是這樣來子的 身為公司的福利委員 每個月都有兩天要拜拜 🙏 是個很虔誠的公司呢！ 但常常會忘記當天要拜拜這件事情（因為用農曆計算日期） 但身為一個工程師 怎麼可以花腦袋記這些事情呢 😧 所以就來製作一個通知機器人吧～ 計畫是按照現行拜拜的日子 來設一個 slack 通知提醒

需要使用到的工具有三個，接著就來一一說明

  1. 農曆計算日期
  2. 在 google spread sheet 上面放程式碼 & 設定排程
  3. 呼叫 slack webhook
...

計算農曆日期

這個部分使用 Intl.DateTimeFormat(區域).format(日期) 要計算農曆的話就是：

  const lunarDay = Intl.DateTimeFormat("zh-TW-u-ca-chinese").format(new Date("2022/12/12")); // '2022年冬月19'

  //這裡我只需要"月"後面的文字
  const [, todayOfLunarCalendar] = lunarDay.split("月"); // 19

  // 做簡單的判斷， 如果今天是 1 或者 15 就是要拜拜的日子
  if(todayOfLunarCalendar === "1" || todayOfLunarCalendar === '15'){
    // 要拜拜， 傳送訊號到slack api
  }

參考來自MDN

使用 Google Sheet 放程式碼與排程

首先，你要先開啟一個新的試算表 上方工具列裡 點擊擴充功能 -> App script 這 script 裡就是放程式碼的地方

  // 大致上與普通寫程式差不多 
  function reminder () {
    // 檢查今天是不是需要拜拜的日子 
    if (todayOfLunarCalendar === "1" || todayOfLunarCalendar === "15") {
      // 呼叫 slack api, url 等等會換成 slack 的 api
      UrlFetchApp.fetch("url", {...config})
      return ContentService.createTextOutput(JSON.stringify({status:200, msg:"success"}))
    }
  }

  // 註一： 在 App Script 裡，要使用 fetch 的話，要用這個 =>  UrlFetchApp.fetch

  // 註二： 要回傳 Response 時，要使用 ContentService.createTextOutput(JSON.stringify(result))

有興趣的話，可以在官方文件裡找一些參考

使用 Slack api 傳送訊息

這裡我使用的是 Slack 的 webhook 先傳送官方文件 首先要先在右上角建立一個你的 Apps 接著點進你的 Apps 裡面後， 選擇『 Add features and functionality 』 點選新增一個 Webhook 到工作區 你會得到一個 URL 大致上長這樣： https://hooks.slack.com/services/T1CXXXXXX/B04DXXXXXXX/gkXXXXXXXXXXXXXXX 你可以直接用 Postman 測試這個 URL 使用參數

  method: POST 
  Header: Content-type: application/json
  body:   {"text":"Hello, World!"}

Google Sheet 設定排程

設定完之後 回到剛剛的 Google Sheet 裡把 slack api 補進去 UrlFetchApp.fetch （可以在程式碼上方欄位找到執行，按下去則可以針對選擇的函式做測試喔！） 確認無誤後，可以選擇左方 sidebar 的鬧鐘圖示（觸發條件） 點選藍色按鈕 新增觸發條件+ 然後調整觸發條件為你需要的觸發時機 例如 這次我需要每週都執行一次 可以選擇 活動來源：時間驅動 => 週計時器 => 每週一 => 上午9點 => 儲存 這樣一來 程式就會自動在每週一上午九點執行一次 如果有錯誤也會通知你 如果是第一次使用這個功能 App Script 會要求你開通授權，按確認之後一切就準備就緒了！
