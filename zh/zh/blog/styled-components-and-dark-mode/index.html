Styled-components 與 暗黑模式
2022-10-26
圖片來源 unsplash

暗黑模式

今天想來分享一下我實作暗黑模式的作法
如果有別的更好的作法 也可以讓我知道（如果有觀眾在看的話

首先 我使用的是 styled-components 這個套件
我覺得很方便的是 可以把樣式寫成元件然後套用進去你想套用的地方

我在包住元件的 Layout 裡面放入一個 ThemeProvider

Layout 範例如下 
// Container.js
import styled, { ThemeProvider } from "styled-components";
import { lightTheme, darkTheme } from "@layouts/theme";

export default function Container({ children, location }) {
  const [theme, setTheme] = useState("dark");
  const isDarkTheme = theme === "dark";
  const toggleTheme = () => setTheme(isDarkTheme ? "light" : "dark");

  return (
    <ThemeProvider theme={isDarkTheme ? darkTheme : lightTheme}>
      <Layout>{children}</Layout>
    </ThemeProvider>
  );
}

這樣一來就可以利用一個 state 去控制模式的切換了

在 styled-components 裡面
經由 ThemeProvider 提供 theme 後
就可以直接在 styled 裡操作 theme 這個變數了～
使用起來很方便

// styled

const styled.div`
  color: ${({theme}) => theme.color}
`

// 👆 在上面的例子裡 theme提供的 darkTheme/lightTheme 的物件
// 物件裡面大致上是這樣子的

lightTheme = {
  color: "#f00037"
}

darkTheme = {
  color: "black"
}
