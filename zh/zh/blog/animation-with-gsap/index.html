參考套件 GSAP
這次主要使用的是 ScrollTrigger 這個插件
除了滑動效果以外
他還有強大的 Tween 跟 Timeline
可以在文件裡面找到這些功能的使用方式
這個套件實在是很厲害

滿簡單可以使用的
需要讓他知道 你想動起來的物件是哪一個

範例如下 
let element = document.querySelector(".animated-box");
gsap.from(element, { opacity: 0, x: 100 });

// 讓這個 .animated-box 元件， 從(from) 透明度0 水平位置 100
// 移動回到 x =0, opacity =1 的階段

可以在使用 from/to/fromTo 來設定動畫的開始狀態與結束狀態
而這次我基本上只有使用到最最簡單的功能

我先測試了一下 使用方式
範例差不多如下的 codepen 可以進去裡面 調整看看各個參數 看動畫會有什麼變化！

scrollTrigger 參數介紹：

  • scrub: boolean/number 動畫需要多久時間跟上滑動距離 如果是 boolean(true) 的話代表，動畫 = 滑動距離
    所以向上滑動 動畫就會倒著動

  • trigger 滾動的基準物件

  • start 兩個詞組成 如："top center" 第一個詞代表 動畫會開始動起來的位置(當基準線碰到這裡的時候) 第二個時是基準線的位置

  • end (基本上與上面相同)

可以在這邊找到更多詳細的說明

如果要在 next 或是 react 這類的框架下使用的話
需要把這個套件用在 useEffect 裡面
除此之外 就是大同小異了

React 範例如下 
// 父元件內
import { gsap } from "gsap";
import { ScrollTrigger } from "gsap/dist/ScrollTrigger";

const container = useRef();
gsap.registerPlugin(ScrollTrigger);
useEffect(() => {
  const element = container.current;

  gsap.fromTo(
    element.querySelector(".target_element"),
    { opacity: 0, y: 10 },
    {
      opacity: 1,
      y: 0,
      scrollTrigger: {
        trigger: element.querySelector(".first"), // 這邊是滑動的觸發元件
        start: "top center",
        end: "bottom center",
      },
    }
  );
}, []);

/*
滑動的觸發元件 就是有點像以這個元件為參考點 去繪製滑動動畫的起始/結束點
*/

return <div ref={container}>/* 這裡面放子元件們 */</div>;

// 我把container 放在 最外層的元件裡 作為ref

// 子原件裡
// 我把container.current當作prop : element傳進來

function Child({ element }) {
  useEffect(() => {
    gsap.fromTo(
      element.querySelector(".animate") // select which element you want to add animation
      //...animation
    );
  }, []);

  return <div className="animate"></div>;
}
