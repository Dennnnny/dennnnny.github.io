等到跑完指令後
專案就可以 run 起來了
輸入npm run develop 就可以在 local 端看到現在的畫面了
預設是在 http://localhost:8000 可以看到剛跑起來的網頁

然後 gatsby 是使用 graphql
可以在 http://localhost:8000/\_\_\_graphql 這裡開啟資料庫

接著就是開發了～

我在這部分一開始的時候 遇到了一點點小問題
主要都是類似

Something went wrong when trying to add the plugins to the project...

or

Error [ERR_PACKAGE_PATH_NOT_EXPORTED] ...

然後發現 我的 node 版本沒跟上 gatsby 的需求
所以有些套件會無法成功的執行
這邊要確保 Node.js (v14.15 or newer)
就沒問題了～
