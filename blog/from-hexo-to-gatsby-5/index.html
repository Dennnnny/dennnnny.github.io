叫做 Utterances

我這邊選擇了 Utterances

步驟也是滿簡單的

首先： 要授權給這個套件
接著： 開一個 repo 來放留言 ex: comments-repo
然後： 可以在這個網頁裡產出設定檔
=> 填入 username/repo-name
=> 選擇文章與留言區的 mapping 方式
=> 選擇主題
最下方會產生一個設定檔
這就是要放進 script 裡面的設定 就會自動產生一個留言區了～～

現在來創一個元件： Comments.js 用來載入這個 script

export default function Comments() {
  const commentsRef = React.useRef(null);
  const theme = useTheme();

  const githubTheme = theme?.type || "dark";

  React.useEffect(() => {
    const script = document.createElement("script");
    script.src = "https://utteranc.es/client.js";
    script.setAttribute("repo", "Dennnnny/dennnnny.github.io.comments");
    script.setAttribute("issue-term", "pathname");
    script.setAttribute("label", "✨💬✨");
    script.setAttribute("theme", `github-${githubTheme}`);
    script.setAttribute("crossorigin", "anonymous");
    script.async = true;

    const COMMENT_REF = commentsRef.current;

    if (COMMENT_REF) {
      COMMENT_REF.appendChild(script);
    }

    return () => {
      if (COMMENT_REF) {
        COMMENT_REF.innerHTML = "";
      }
    };
  }, [githubTheme]);
  return <div ref={commentsRef} />;
}

用 useEffect 來設定 script
因為我有用暗黑模式 所以要在切換模式的時候 重新載入一次 留言區的模式設定
如果你沒有的話 可以直接把主題寫進 script 不用動態的
這樣就算是 完成了留言元件

只要把它放到文章裡面就可以看到他出現在文章的下方了～～ 算是滿簡單的功能
但是非常實用 ＸＤ
