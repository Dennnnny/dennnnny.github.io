Start to use Gatsby to build blog

I used hexo to write blog
hexo let me build a blog in very short time, like really fast

you can find many themes in official website or in gatsby community
it's very convenient ot use themes
but, sometimes I thought those themes should had some features, ans it's not easy for me to change themes of hexo

Then, I found Gatsby.
It has more flexibility for me
cause I use React for work, and so is Gatsby
so, it's easier for me to build theme by myself
though, it means I would pay more attention on building themes
yet, I actually enjoy the development process.

there are some other framework which also based on React
such as next.js or remix.js
I think they can achieve the same level maybe even more powerful.
but I think I only need some static website for my blog. so Gatsby is enough for this project
That's it, I decide to use Gatsby to build a blog

I will try to share some obstacles that I deal with when I developed this project
