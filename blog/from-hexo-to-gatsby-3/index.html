搭配 styled-components

由於這是我自己現在用的最習慣的樣式寫法
所以這次在開發的過程中 就直接選擇了 styled-components
這次主要會在分享之前還沒用過的 global-theme 用法
因為想要做 darkmode
就會需要用到一個可以切換的顏色對照的字典之類的參考

我會使用一個 style-config.js
來設定一些可能會共用的樣式
例如各種顏色 也許字體大小/粗細 也都可以在這裡設定
然後 export 出去
在其他頁面裡可以直接抓變數顏色
以後想要改整體色系的話
就可以藉由更改 config 裡面的設定 就可以套到所有的地方
很方便～～

在其他元件裡使用的話 就會像是這樣

//Component

import { colors } from "....";

const StyleLayout = styled.div`
  background: ${colors.white};

  @media ${device.tablet} {
    background: ${colors.blue};
  }
`;

dark mode

來做黑暗模式
首先
我在/layouts 裡面 放了一個 theme.js 檔案 這裡分別是 light mode 跟 dark mode 時要用到的參考顏色之類的

// theme.js
export const lightTheme = {
  type: "light",
  bgColor: backgrounds.bgLight,
  ...
};
export const darkTheme = {
  type: "dark",
  bgColor: backgrounds.bgDark,
  ...
};

然後 要在最外層包起來的地方 使用 Theme
我這邊是用在自製的 Container 裡
這個 Container 會包在所有的頁面外面 所以很適合～～

ThemeProvider 是 styled-components 提供的元件 用來切換主題用的
使用 useState 來做模式的切換
{ lightTheme, darkTheme } 就是在 theme.js 裡面做的設定

// Container.js
import styled, { ThemeProvider } from "styled-components";
import { lightTheme, darkTheme } from "@layouts/theme";

export default function Container({ children, location }) {
  const [theme, setTheme] = React.useState("dark");
  const isDarkTheme = theme === "dark";
  const toggleTheme = () => setTheme(isDarkTheme ? "light" : "dark");

  return (
    <ThemeProvider theme={isDarkTheme ? darkTheme : lightTheme}>
      <Layout>...children</Layout>
    </ThemeProvider>
  );
}

以上大致上就是這次第一次做暗黑模式的作法
我覺得還不錯～
在一個 config 裡面設定共用樣式也是很方便管理
